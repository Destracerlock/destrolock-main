### Loading the Chrome extension

1. Open `chrome://extensions` in Chrome.
2. Enable Developer mode.
3. Choose **Load unpacked** and select the `repos/chrome-timer-extension`
   folder.

Press **Start** in the popup to begin the two minute timer. A notification will
appear when time is up.
